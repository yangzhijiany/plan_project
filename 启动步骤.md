# 🚀 LLM Task Planner 启动步骤

## 第一步：启动后端服务器

打开终端，执行以下命令：

```bash
cd /Users/jerry.yang/plan_project/backend
source venv/bin/activate
uvicorn main:app --reload
```

**等待看到以下信息表示启动成功：**
```
INFO:     Uvicorn running on http://127.0.0.1:8000
INFO:     Application startup complete.
```

✅ **后端运行在：http://localhost:8000**

---

## 第二步：启动前端服务器

**打开新的终端窗口**（保持后端运行），执行以下命令：

```bash
cd /Users/jerry.yang/plan_project/frontend
npm run dev
```

**等待看到以下信息表示启动成功：**
```
  VITE v5.0.8  ready in 500 ms
  ➜  Local:   http://localhost:5173/
```

✅ **前端运行在：http://localhost:5173**

---

## 第三步：访问应用

在浏览器中打开：**http://localhost:5173**

---

## ⚠️ 注意事项

1. **必须同时运行后端和前端**
2. **需要两个终端窗口**（一个后端，一个前端）
3. **后端必须在前端之前启动**
4. **不要关闭终端窗口**，关闭终端会停止服务

---

在对应终端按 `Ctrl + C` 即可停止服务

## ❓ 遇到问题？

### 后端启动失败
- 检查虚拟环境是否激活：`which python` 应该显示 `venv/bin/python`
- 检查依赖是否安装：`pip list`
- 检查 API Key：`cat backend/.env`

### 前端启动失败
- 检查依赖是否安装：`ls frontend/node_modules`
- 检查 Node.js 版本：`node --version`（需要 16+）
- 重新安装依赖：`cd frontend && npm install`

### 前端无法连接后端
- 确认后端正在运行（访问 http://localhost:8000）
- 检查浏览器控制台错误信息
- 确认两个服务都在运行
